Campus Events – Web Tabanlı Etkinlik Yönetim Sistemi

Bu repo, AMPS (Apache–MySQL–PHP–phpMyAdmin) ortamında çalışan Campus Events web tabanlı etkinlik yönetim sistemi projesini içermektedir.
Proje, kampüs içi etkinliklerin rol tabanlı olarak yönetilmesini amaçlamaktadır.

Uygulama, AMPS kurulum dizini içindeki www klasörüne yerleştirildikten sonra aşağıdaki adres üzerinden çalıştırılır:

http://localhost/campus_events/index.php

Nasıl Çalışır?

Kullanıcı tarayıcı üzerinden sisteme erişir. Apache gelen isteği karşılar, PHP dosyaları sunucu tarafında çalışır, gerekli veriler MySQL veritabanından alınır ve sonuçlar kullanıcıya web arayüzü olarak sunulur.

Campus Events

Sistem 3 farklı kullanıcı rolü ile çalışmaktadır:

Student: Etkinliklere kayıt olur, kaydını iptal eder ve etkinlik sonrası geri bildirim verir.

Organizer: Etkinlik oluşturur ve yönettiği etkinlikleri görüntüler.

Admin: Yeni kullanıcı tanımlar, etkinliklere organizer ataması yapar/değiştirir, sistem loglarını ve en popüler 5 etkinliği görüntüler.
